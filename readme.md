# Lets Golang Framework

Lets adalah framework yang menggabungkan beberapa framwork untuk bootstraping servis.

**Kelengkapan:**
- Saga (Orchestration & Choreograpy) with API-Driven and Event-Driven
- HttpFramework
- RabbitMQ
- GRPC
